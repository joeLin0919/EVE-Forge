# 贡献指南

感谢您对 EVE-Forge 项目的关注！我们欢迎各种形式的贡献。

## 开发环境设置

1. 安装 JDK 21
2. 安装并启动 Redis 服务
3. 安装并配置 MySQL 数据库
4. 克隆项目仓库
5. 根据 `.env.example` 创建并配置环境变量

## 提交代码

1. Fork 项目
2. 创建功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 创建 Pull Request

## 代码规范

- 遵循项目现有的代码风格
- 添加适当的注释
- 确保代码测试通过

## 报告问题

- 使用 Issues 功能报告 bug
- 清楚描述问题现象和重现步骤
- 包含错误日志和环境信息

## 提出功能建议

- 在 Issues 中详细描述功能需求
- 解释该功能的价值和应用场景
